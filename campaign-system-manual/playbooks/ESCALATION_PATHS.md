# Escalation paths — Pass 4

**Context:** The Campaign Operating System routes decisions through the lowest qualified approver (LQA) and explicit break-glass rules. This document is **not** a substitute for law, treasurer sign-off, or `MANUAL_INFORMATION_REQUESTS_FOR_STEVE.md`.

**Public language:** Workbench, Pathway, Campaign Companion — not “AI.”

| Trigger | First responder | When to include CM | When to include owner / counsel | Log / artifact |
|--------|------------------|-------------------|--------------------------------|----------------|
| PII exposure or misfire | data steward + compliance-adjacent lead (per policy) | always | PII to minors, bulk export, press-adjacent | ticket + access log note |
| Finance amount above threshold (internal) | treasurer path | if program tie-break | if political spend / crisis comms | `FinancialTransaction` **CONFIRMED** status |
| Proposed commission / ambassador pay | n/a **until** treasurer + counsel | inform only | O + T + C only to activate | 3G grass roots legal block |
| Public statement error or leak | comms + MCE | if schedule / turf impact | if legal contrast or defamation risk | retraction SOP + counsel |
| Voter file export request | data steward | M for roster alignment | O for break-glass export | `MANUAL_INFORMATION` §6 |
| County / party / NAACP public claim not verified | relationship steward | M + NDE for words | C if org identity | no invented branches |
| Youth / campus / worship visit edge | program lead + advance | M | C + O for minors or worship sensitivity | YOUTH/FAITH hooks |
| Election Day public incident | CM + comms | n/a | O (+ C) for media | ED runbooks (when published) |
| System abuse / access creep | admin / operator (when exists) + CM | roster review | O for revoke | Workbench access policy |

**Chain:** *peer lead → program lead → campaign manager → owner* for operations; *treasurer* parallel for money; *counsel* for legal/claim edges.

**Crisis compress:** In the final 10 days, owner may set a standing **Crisis response** list (see `APPROVAL_AUTHORITY_MATRIX.md` and `MANUAL_INFORMATION_REQUESTS_FOR_STEVE` §5, §8, §15).
