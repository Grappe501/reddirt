## T1. Campaign-wide morning brief

- **Header:** Date · scope (all-staff) · owner (CM or delegate)  
- **Objectives:** Today / week / month (one line each)  
- **Key** **movement:** Honest count-based signals only (P0, aging buckets — no names in general brief)  
- **What** **changed** vs yesterday  
- **Priority** **tasks:** Top 3  
- **Follow**-**up:** By role, not by voter name  
- **Approvals:** List types needing LQA / MCE / counsel per matrix  
- **Tangible:** 15m open sweep; 1h triage block  
- **Risks** / **DNT:** Unapproved contrast; high-impact publish without 5G triage  
- **Ask** **Kelly** **prompts:** (internal) “Summarize suggestion counts by type” not public voter claims
