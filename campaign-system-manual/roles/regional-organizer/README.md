# Role manual — Regional organizer

## 1. Role purpose
Allocate **playbooks**, **time**, and **honest** peer **tables** across **counties** in a **campaign** **region** (8 OIS slugs).

## 2. Why this role matters
**Stops** **NWA** vs **River Valley** **disconnection** in **message** and **event** timing.

## 3. Where this role sits
Between **state** OIS narrative and **county** leaders; **not** **voter** **row** access for **fun**.

## 4. Who this role serves
**County** leaders and **CM** on **capacity** truth.

## 5. Who supports this role
**Field** mgr (line), **data** (aggregate **honesty**), **comms** (regional line of week if used).

## 6. Dashboard used
OIS ` /organizing-intelligence/regions/...` + **workbench** for **triage**; **not** a second **VAN**.

## 7. Manual sections
Ch. 6, 11, 12, 21, **regional** **reports** in `docs/`.

## 8. First 24 hours
**2** P0 **county** **hot** spots; **1** P0 to **comms** if **NWA** / **peer** line wrong in **OIS** **demo** vs **reality** review.

## 9. First 7 days
**Roster** of **county** leaders health; **event** **collision** check across county lines.

## 10. First 30 days
**30d** **resource** plan (time, not cash unless RACI) with **field**+**CM**.

## 11. Daily workflow
**GOTV**-adjacent; else **2** **huddles**/week w/ subregions.

## 12. Weekly workflow
**State**+**OIS** **read**; **CM** **sync**; **narrative** **NDE** if **wires** in **product**.

## 13. KPIs
**Gap** closure across **peers** (safe, aggregate); **error** in **OIS** **numbers** **found**; **0** PII in **email** out.

## 14. Workbench tasks
`WorkflowIntake` w/ `countyId` in **region**; `CampaignTask` for **tours**; **comms** **plan** w/ `source` **intake** if you originate.

## 15. Approval authority
**Rebalance** **field** time; not **$** w/o fin; not **messaging** on **opposition** w/o comms+compliance.

## 16. Training modules
OIS **demo** **honesty**; **NWA**-vs-central **taxonomy**; **ethics** on **comparing** **counties** in **public** without **shaming** individuals.

## 17. Tools used
OIS, **county** **admin** if granted, **GOTV** in **ED** window, **narrative** **admin** if assigned.

## 18. Common mistakes
**Vanity** **leader**boards; **bypassing** **county** on **hiring** **locals**; **OIS** as **VAN** on phone to volunteers.

## 19. Escalation path
**Field** mgr; **CM**; **owner** for **$**+**hiring**; **data** for **mismatch** in **file**-driven number.

## 20. Growth path
**Deputy** **state**; sideways to **comms** if loves **Narrative** + **OIS** together.

## 21. Election Day
**Roving** support per **RACI**; **comms** **single** **voice** on **outcome** **claims** **post**-polls.

## 22. Missing system features
**One** `region` object tying **8** OIS to **file**-rolled **KPIs** w/o manual CSV.

## 23. Current readiness level
**4** OIS **read**; **3** for **unified** **rollups** from **P5+GOTV** without **gaps** in product.
