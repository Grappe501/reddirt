# Role manual — City captain

## 1. Role purpose
Municipal **rollups**: recruit/align **precinct** + **neighborhood** leads; event density w/ event lead.

## 2. Why this role matters
Cities are **hubs** for campus, **hosts**, and **local** **earned** **media** opportunities.

## 3. Where this role sits
Between **county** and **precinct**/**block**; **OIS** when city routes exist (today **0–1** in routes).

## 4. Who this role serves
**County** strategy and **local** org culture.

## 5. Who supports this role
**County** leader, **regional** organizer, **field** mgr, **V.C.**

## 6. Dashboard used
**Planned** `.../cities/...` OIS; now **county** + **tasks** + huddle docs.

## 7. Manual sections
Ch. 10–11, 22, lifecycle 4, `visibility` **workflow**.

## 8. First 24 hours
Name **bench** for **subcaptains**; **1** P0 call for **empty** city if needed.

## 9. First 7 days
**2** **sister**-city **gap** **comparisons** in **OIS** **honest** data only.

## 10. First 30 days
**30d** event + **GOTV** plan **handshake** w/ event+GOTV.

## 11. Daily workflow
**GOTV**-adjacent: **2** **push** **blocks**; else **2–3** **week**.

## 12. Weekly workflow
**County** **sync**; **field** **sync**; **OIS** **read** for **gaps** not **vanity** numbers.

## 13. KPIs
**Captain** **bench** depth, **event** **fill** **proxy**, **%** **precs** w/ a **P5** **anchor**.

## 14. Workbench tasks
`CampaignTask` for city-**scoped** work; intakes w/ `countyId` that apply to you via policy.

## 15. Approval authority
**Local** **spend** only per **RACI**; **comms** through **comms** lead on **municipal** **claims**.

## 16. Training modules
Municipal **etiquette**; **safety** on campus if applicable; **OIS** **honest** **missing** data.

## 17. Tools used
(`Future`) OIS **city** **shell**; **events**; **county** **admin** if granted.

## 18. Common mistakes
**Overclaiming** **organizing** on **OIS** **demo** tiles; **bypassing** **county** on **messaging** line.

## 19. Escalation path
County **leader**; **regional** for **funding**; **CM** for **turf** **disputes** between **subcaptains**.

## 20. Growth path
**County** leader or **regional** **deputy**; sideways to **event** lead in **metro**.

## 21. Election Day
**City** **GOTV** **bridge**; **escalation** to **GOTV** for **VBM**; **safety** to **compliance** on **incidents** at **sites** you named.

## 22. Missing system features
OIS **city** **URL**+**data**; **chips** w/o PII; **NWA**-style **cluster** in **OIS** if multi-**muni** county.

## 23. Current readiness level
**2** product; **3** w/ SOP+county+field alignment.
