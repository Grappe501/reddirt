# Role manual — Precinct captain

## 1. Role purpose
**Aggregate** **turf** health for a **precinct** **key**; **not** a public list browser.

## 2. Why this role matters
**Electoral** work concentrates at **precinct**; connects **P5** to **GOTV**.

## 3. Where this role sits
Under **city**/field; over **P5** teams in **that** **precinct** where assigned.

## 4. Who this role serves
**Voters** (through programs), **field** and **GOTV**.

## 5. Who supports this role
**City** captain, **GOTV** lead, **data** (keys only).

## 6. Dashboard used
**Planned** OIS list/chip; now **tasks** + OIS read **aggregates** if allowed.

## 7. Manual sections
Ch. 10–11, 22, sign + **visibility** workflow.

## 8. First 24 hours
**Gap** list; **1** P5/team check-in.

## 9. First 7 days
**%** of **P5** in prec with **GOTV** plan **touch**; **pair** w/ sign captain.

## 10. First 30 days
**Roster** of **subcaptains**; **conflict** map for **GOTV**.

## 11. Daily workflow
**GOTV** window **daily**; else **2–3**/week.

## 12. Weekly workflow
**City** and **GOTV** huddle; **OIS** **read** w/ “demo vs live” care.

## 13. KPIs
Coverage % (definition fixed), **drop**-off repair.

## 14. Workbench tasks
`CampaignTask` for prec **gap**; not **voter** row tasks on personal phone in public.

## 15. Approval authority
**Field**-scoped asks; not **contrast** ad copy.

## 16. Training modules
**DATA-1** honest limits; public map **limits**.

## 17. Tools used
Task lists; (future) **OIS** prec route.

## 18. Common mistakes
Assuming `VoterRecord` on web for volunteers; **doxing** in chat.

## 19. Escalation path
City captain → field mgr; **GOTV** for **VBM** line issues; **compliance** if cited.

## 20. Growth path
City captain; to **data**-adjacent if loves lists (under policy).

## 21. Election Day
Own **prec** **GOTV**; **triage** to **GOTV** lead.

## 22. Missing system features
OIS `precinct` product route; **chips** w/o addresses on public OIS need **policy** before launch.

## 23. Current readiness level
**2** route+data; **3** w/ SOP+tasks in **GOTV** window.
