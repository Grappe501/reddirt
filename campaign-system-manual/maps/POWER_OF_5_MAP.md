# Power of 5 map

**Conceptual stack (P5 plan):** Individual → Power team (~5) → Precinct (aggregate) → City → County → Region → State.

**Public entry:** ` /onboarding/power-of-5`

**Data objects (illustrative in plan):** `PowerNode`, `PowerTeam`, `PowerInvite`, `OrganizingActivity` — **Prisma** alignment in future packets.

**Dashboards:** P5 **panels** appear on OIS/county/region in **demo** or partial form — verify in UI before “production” claims (see P5 integration docs).

**Privacy:** Roster **consent-scoped**; no public shaming; voter file is **not** a volunteer browse surface.
