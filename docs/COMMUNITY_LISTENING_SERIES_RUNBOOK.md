# Community Election Series & listening sessions — field runbook (Pass 1)

**Scope:** In-person and hybrid **listening** and **civic** sessions (e.g. election integrity dialogues, partner-hosted forums). Aligned with events like `town-hall-garland-library-*` in `src/content/events` where **host** sets **ground rules** (e.g. not a campaign rally).

**Last updated:** 2026-04-27.

---

## 1. Roles

| Role | Responsibility |
|------|----------------|
| **Campaign coordinator** | Master calendar, staff coverage, comms line for RSVPs, conflict resolution. |
| **Event coordinator (session lead)** | Day-of: doors, PPE, timekeeping, name tags, **ground rules** read aloud. |
| **Facilitator** | Neutral time/steps; not a **stump** block unless a **separate** candidate time is agreed. |
| **Notetaker** | **Themes** (de-identified), **not** PII in public-facing summary without consent. |
| **Comms** | Photo/video policy, **one** public recap, press **embargo** if any. |
| **Volunteer captain** | Greeting, sign-in, water, accessibility. |

## 2. Pre-session checklist (T-7 / T-3 / T-1)

- [ ] **Venue** contract + **ADA** and **parking** note on invite.  
- [ ] **Host** (PARTNER) name on **all** public materials.  
- [ ] **RSVP** path (form or Google) + cap.  
- [ ] **Ground rules** (respect, time limits, no harassment).  
- [ ] **Emergency** contact on site.  
- [ ] For **Braver Angels**-style: confirm **not** a **rally** per host agreement.  

## 3. Day-of flow (suggested 90 min)

1. **Welcome** (5) — host + coordinator.  
2. **Why we’re here** (5) — process, not debate outcome.  
3. **Rounds** (50) — small groups or full room per format.  
4. **Report-back** (20) — themes, **no** **forced** **consensus**.  
5. **Next steps** (10) — optional signup; **not** a hard donate pitch unless on-brand.  

## 4. Tools (build targets for Pass 3)

- **Calendar** — `Google` / `Campaign` calendar; optional link to `RedDirt` admin /events.  
- **Email** — SendGrid or Mailchimp **segment**: “**Listening** **alumni**.”  
- **Volunteer** — `VolunteerForm` / CRM tag `listening_session`.  
- **Tasks** — admin **tasks** board: **materials, AV, name badges**.  

## 5. KPIs (report weekly)

| Metric | How |
|--------|-----|
| **Sessions held** | Count |
| **Attendees (check-in)** | Count |
| **Counties** | List |
| **Themes captured** | 3–5 bullets per session |
| **Media** | # clips / articles |
| **Follow-up** contacts | # |

## 6. Data & privacy

- **Do not** put **sensitive** **voter** stories in public DB without **release**.  
- **PII** → admin-only.
