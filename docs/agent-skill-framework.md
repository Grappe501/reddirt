# Campaign agent — skill framework (SKILL-1) (RedDirt)

This document defines the **formal** **capability** **domains** for a **governed** “digital **Campaign Manager**” (assistant + RAG + ops) that **sits** **on** the **same** **rails** as `campaign-engine`, workbenches, and **queue-first** comms. It is an **architecture** artifact, **not** a product promise that every model call today implements all rows.

**Cross-ref:** `src/lib/campaign-engine/skills.ts` (`AgentSkillDomain`, …) · `docs/ai-agent-brain-map.md` · `docs/ai-integration-matrix.md` · `docs/campaign-brain-alignment-foundation.md` · `docs/user-scoped-ai-context-foundation.md` · `docs/agent-knowledge-ingest-map.md` (ingest order)

**Governance invariants (unchanged from ALIGN-1 + handoff):** no **autonomous** sends, no **hidden** **authority** **escalation**, **human** on **governance** **boundaries**, **queue-first** where policy requires.
