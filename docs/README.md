# Red Dirt — campaign & brand documentation

This folder is the **source of truth** for vision, philosophy, brand, and how the public site should evolve. Update these docs as the movement grows; the website should be changed to match what’s locked here.

## How to use

| You need… | Open… |
|-----------|--------|
| Why we exist and how we think | [`philosophy/`](philosophy/README.md) |
| Public narrative & manifesto | [`narrative/`](narrative/README.md) |
| Visual system & tokens for build | [`brand/`](brand/README.md) |
| Features the site must support (forms, pages, IA) | [`platform/`](platform/README.md) |

## Conventions

- **Philosophy** = durable ideas (update when strategy shifts).
- **Narrative** = copy blocks and story structure (update per election cycle or launch).
- **Brand** = tokens and rules for designers and engineers (version when you intentionally rebrand).
- **Platform** = product/IA checklist tied to organizing tools (update when you add programs).
