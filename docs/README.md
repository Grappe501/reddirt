## Conventions

- **Philosophy** = durable ideas (update when strategy shifts).
- **Narrative** = copy blocks and story structure (update per election cycle or launch).
- **Brand** = tokens and rules for designers and engineers (version when you intentionally rebrand).
- **Platform** = product/IA checklist tied to organizing tools (update when you add programs).
