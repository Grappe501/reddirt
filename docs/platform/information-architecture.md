# Information architecture & conversion paths

Checklist for the public site. Update as programs launch.

## Primary navigation (suggested)

- Home  
- About / Movement  
- Issues *(optional anchor pages)*  
- **Direct Democracy** or **People Power** *(see [`civic-participation.md`](civic-participation.md))*  
- Events  
- Volunteer / Get involved  
- News / Updates  
- Donate *(as allowed)*  
- Contact  

## Direct democracy / civic power area

**Dedicated page(s)** explaining:

- Referendum / initiative role in Arkansas  
- The commitment network (what signing up means; privacy; legal disclaimers)  
- Civic education: how the process works  
- Optional: youth participation exploration (informational)  
- Optional: workforce / labor hub *(if built)*  

## Homepage integration

- Section introducing the **referendum / people’s voice network**  
- CTA: **“Commit to protecting the people’s voice”** (exact wording after legal/comms review)  
- Secondary pathway: *“I want to help protect the people’s vote”*

## Forms — suggested fields / opts-in

- Name, email, phone, ZIP/county *(organizing)*  
- Optional checkbox: **“I commit to signing referendum petitions when needed”** *(wording TBD with counsel)*  
- Interest tags: volunteer, host event, community leader, civic action alerts  

## Conversion pathways (plural)

1. Join / volunteer  
2. Stay connected (email/SMS)  
3. **Civic action / referendum network**  
4. Donate *(if live)*  
5. Learn (civic education content)

## Related neutral labels (IA alt)

If “Direct Democracy” is sensitive in a given context, use:

- **Civic Power**  
- **People’s Process**

See also [`neutral-module-spec.md`](neutral-module-spec.md) for modular product description.
